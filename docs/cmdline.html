14 Command line switches

Note: All switches are case-sensitive.

SwitchAction
--log-to-fileTurn on debug mode.
--group-name=name_of_groupSet "name_of_group" group of dictionaries as current for main window.
--popup-group-name=name_of_groupSet "name_of_group" group of dictionaries as current for popup window.

Switches "--group-name" and "--popup-group-name" can affect to already loaded GoldenDict instance. Also command line can contain word which will be transferred to GoldenDict to translate.
